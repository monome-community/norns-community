## description

 dual async start-stop loop recorder. controls for rate and volume per voice.
 
```
K2/K3 = toggle rec/play
K1+K2/K3 = stop
E2/E3 = change param value
E1 = change selected param 
```

## install

from maiden type
`;install https://github.com/tehn/ash`
